# Heyvin AI - Private Sovereignty & Focus Workspace

Heyvin is an offline-first, zero-telemetry personal success auditor and mental sanctuary crafted for ambitious young women navigating high domestic friction, caretaking constraints, or conservative family settings. Designed to help students and professionals protect their study hours, practice boundary setting, and capture university degrees or independent careers with absolute confidentiality.

---

## 🌸 Core Mission

In many traditional households, young women bear disproportionate domestic chores, unexpected guest hosting duties, and caretaking tasks that fragment their "Focus Capital." **Heyvin** is engineered to act as a mathematical auditor of this daily friction while serving as a private, secure emotional buffer.

Every feature of Heyvin works under a **Strict Sovereignty Policy**: no logs are sent to the cloud, no telemetry is tracked, and everything remains nested in your browser’s physical cache.

---

## ✨ Immersive Feature Suite

### 1. 🛡️ Ultimate Stealth Cover Switch (StudySync)
If unannounced house guests or family members enter your study space, triple-clicking the top Logo or hitting an emergency hotkey instantly morphs the interface into **StudySync**—an unassuming, static academic course directory (e.g., "Intro to Statistical Models" or "Syllabus Guides"). Your actual study goals and logs remain hidden until you safely enter your stealth toggle.

### 2. 📊 Autonomy & Workload Auditing
- **Focus Capital Simulator**: Calculate and preview how your local household chores, unexpected summons, and noise levels affect your actual daily academic pacing.
- **Sovereignty Score**: Track your daily and weekly autonomous ratios—mathematical modeling of your time sovereignty.
- **Pattern Prediction Engine**: Identify which household routines consistently overlap with your optimal study sweet-spots.

### 3. 🧠 Therapeutic Coping & Sanctuary Exercises
- **Heyvin AI Journal**: Safely dump anxious thoughts in a private localized ledger and receive reassuring therapeutic framing guidelines.
- **Boundary Rehearsal Suite**: Interactive sandbox to practice boundary phrasing with relatives before sudden home interruptions.
- **Support Shoutbox**: A pseodonymous, real-time message stream to coordinate work shifts and send study support tokens to other female scholars globally.

### 4. 🔑 Multi-Channel Safe Onboarding
Supports fast, encrypted pseudonym logins or physical **Google Sign-In OAuth** credentials for fast university/college workspace connections.

## 🛠️ Technology Stack & Architecture

- **Frontend**: React 18+ (Vite), TypeScript, Tailwind CSS, Lucide Icons, Motion (Framer).
- **Backend API Server**: Node.js & Express, bundled with `esbuild`.
- **Privacy Model**: Fully decentralized local storage caches combined with secure sandboxed server proxies.

## 🚀 Quick Local Development Setup

### 1. Prerequisites
Ensure you have Node.js (v18+) and npm installed.

### 2. Installation
Install all base project dependencies:
```bash
npm install
```

### 3. Environment Configuration
Create a local `.env` file referencing the templates provided in `.env.example`:
```env
# Gemini API client for therapeutic framing logs
GEMINI_API_KEY=your_gemini_key

# Google OAuth tokens for safe login integration
GOOGLE_CLIENT_ID=your_client_id
GOOGLE_CLIENT_SECRET=your_client_secret
```

### 4. Directing Development Servers
To boot the full-stack server under the required port config (`3000`):
```bash
npm run dev
```
Open **http://localhost:3000** in your browser.

## 🔒 Security Commitments

1. **Local-First Proxy Architecture**: Your study plans, weekly reports, and journaling logs reside on your physical computer.
2. **Zero Metadata Leaks**: We do not leverage pixel markers, cookies, third-party analytics scripts, or active web telemetry.
3. **Emergency Bypass**: Active hotkeys instantly scrub ephemeral states on the fly should sudden screen inspections occur.
